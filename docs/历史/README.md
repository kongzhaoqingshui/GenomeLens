## 归档原则

- 文档一旦其描述的工作整体落地、或被新文档取代，移入本目录。
- 归档文档保持原样，不再随代码更新；如需了解最新状态，以生效文档为准。
- 二进制归档文件仅作历史留存，不进入常规文档索引的 Markdown 链接检查范围。
