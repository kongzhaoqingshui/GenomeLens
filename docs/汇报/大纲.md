# PPT汇报大纲

## 一、开场与背景

### P1 - 封面
- 标题：GenomeLens — 面向 Windows 的本地比较基因组学分析平台
- 副标题：JCVI 共线性分析平台的 MVP 设计与实现
- GitHub 仓库链接：`https://github.com/nhAirsy/GenomeLens`
- 日期：2026.6.26

### P2 - 研究背景
- 比较基因组学、共线性分析的简单介绍

- 比较基因组学常用库 JCVI 的优势与痛点
  - 比较基因组学常用 JCVI(很强大)，但命令复杂、依赖多、Windows 环境配置困难、内部使用Linux指令
  - 研究者需要反复跑多个脚本、手动拼接结果
  - 现有方案多为 Linux/服务器端，本地 Windows 用户门槛高

> 由此引入我们“需要”一个 Windows 上本地、轻量、快捷的 JCVI 全链路分析平台，乃至比较基因组学分析平台

### P3 - 项目目标(MVP)
- 在 Windows 上本地、轻量、快捷地完成 JCVI 共线性分析
- 提供「一站式工作流」和「可编排子模块」两种互补用法
- 统一任务协议、统一结果资产、未来可支持多引擎热插拔、AI Agent、深度学习的表型分析与比较基因组结合...

## 六、总结与展望

### P - MVP 版本的缺点

- 尚未完全吃透 JCVI 的所有能力
- 未尝试接入其他分析引擎
- HAIant 插件侧、GUI 支持未完全成熟
- 算法优化、性能优化、可视化优化仍有空间，缺乏算法工程深度

### P - 展望
- 继续完成 GenomeLens 平台的开发，做到真正的 Windows-first 本地、轻量、易用、高效、高保真的比较基因组学分析平台
- 深入算法领域，在算法工程、深度学习、AI Agent 等方面进行探索，寻求“万物皆可比较组学”的境界。
