# GUI 构建与运行

> 本文件说明如何在本地把 `gui/tauri/` 跑起来，以及如何调试构建。

---

## 1. 环境依赖

### 1.1 必需

- **Rust**：通过 [rustup](https://rustup.rs/) 安装，Tauri v2 需要较新版本。
- **Node.js**：≥ 18（推荐 LTS）。
- **pnpm**：`npm install -g pnpm`。
- **genomelens conda 环境**：Python 3.12，`platform/` 与 `engines/jcvi/` 已做 editable install。

### 1.2 可选但推荐

- **WebView2 Runtime**：Windows 10/11 通常已自带；若 Tauri 窗口空白，请检查并更新。
- **Visual Studio Build Tools**（Windows）：Tauri 编译某些 Rust 依赖时可能需要 C++ 工具链。

---

## 2. 初始化

```powershell
# 1. 确保在正确的 conda 环境
conda activate genomelens

# 2. 进入 GUI 目录
cd gui/tauri

# 3. 安装前端依赖
pnpm install

# 4. 安装/更新 Rust 目标（Tauri 需要）
rustup target add wasm32-unknown-unknown
```

---

## 3. 本地开发运行

```powershell
cd gui/tauri
pnpm tauri dev
```

这会同时启动：

- Vite 前端 dev server（HMR）
- Tauri Rust 后端（cargo）
- 一个桌面窗口

首次运行会编译 Rust crate，耗时较长；后续启动较快。

---

## 4. 调试构建

```powershell
cd gui/tauri
pnpm tauri build --debug
```

产物在 `gui/tauri/src-tauri/target/debug/bundle/`（或类似路径），可直接运行以验证打包行为。

> 注意：`--debug` 构建仅用于本地调试，**不用于分发**。

---

## 5. 定位 `genomelens` 可执行文件

GUI 通过 Tauri Command / Sidecar 调用 `genomelens`。开发阶段推荐按以下优先级定位：

1. **conda 环境 PATH**：确保 `genomelens` 在 `genomelens` 环境的 PATH 中。
2. **editable install**：`pip install -e ./platform` 后，`genomelens` 入口脚本会出现在 conda 的 Scripts 目录。
3. **显式配置**：在 GUI 设置中允许用户指定 `genomelens` / `jcvi-genomelens` 绝对路径。

验证：

```powershell
conda activate genomelens
which genomelens        # Git Bash
genomelens --version
```

## 7. 常见问题

### 7.1 `pnpm tauri dev` 卡在编译

- 第一次编译 Rust crate 需要几分钟，请耐心等待。
- 确保 Rust 工具链完整：`rustup update`。

### 7.2 窗口空白或白屏

- 检查 WebView2 是否安装：Windows 设置 → 应用 → WebView2 Runtime。
- 查看 Rust 后端日志，确认前端 dev server 地址正确。

### 7.3 调用 `genomelens` 报“找不到命令”

- 确认 conda 环境已激活。
- 确认 `platform/` 已 `pip install -e .`。
- 在 GUI 设置中显式指定可执行文件路径。

### 7.4 前端热更新不生效

- Vite HMR 通常正常；若失效，尝试 `pnpm tauri dev` 重启。
- 检查 `vite.config.ts` 中的 `server.hmr` 配置。

## 8. 目录说明

```text
gui/tauri/
├── package.json            # Node 依赖与脚本
├── vite.config.ts          # Vite 配置
├── tailwind.config.js      # Tailwind 配置
├── tsconfig.json           # TypeScript 配置
├── src/                    # React 前端源码
└── src-tauri/
    ├── Cargo.toml          # Rust crate
    ├── tauri.conf.json     # Tauri 应用配置
    └── src/
        └── main.rs         # Rust 入口
```
